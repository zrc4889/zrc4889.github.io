博客探索的坎坷之路

发表于 2024-08-22 分类于 随笔

这几天时常会因为捣鼓自己的博客以致于荒废了其他事务。

事实上，这方面相关的技术确实是难度高且耗时大的。

所以对于博客，我衷心建议大家专注于写作，而非花里胡哨的功能

有求必应，有需求再去添加功能。

其实从小开始我也一贯如此。

以前探索 Python 搭建聊天室，了解到网络编程，IIS 探索过。

后来自己“速成” HTML、CSS、Javascript 手工写了一个网站（现在不知道还在不在），那已经是六年级的事情了。

遇到一些数学难题也会所谓“追根溯源”。

但是，按照“儒释道”的原理，人生中其实还有许许多多类似于此纷繁复杂的事情，需要我们“拿得起、放得下”。

共勉。
